# User Management Service

Simple API for registering and fetching a user from a database.


HLD 
------

> **Architecture:** N-tier

### Use Cases
	
  - Register User
  - Get User by ID

> **Swagger Url:** http://localhost:8080/api/swagger-ui/index.html
